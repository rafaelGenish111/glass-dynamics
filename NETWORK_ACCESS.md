# גישה לאפליקציה מרשת מקומית

## הגדרה

האפליקציה מוכנה לגישה מרשת מקומית! כל מה שצריך לעשות:

### 1. מצא את כתובת ה-IP שלך

**ב-Mac/Linux:**
```bash
ifconfig | grep "inet " | grep -v 127.0.0.1
```

**ב-Windows:**
```bash
ipconfig
```
חפש את `IPv4 Address` תחת `Wireless LAN adapter` או `Ethernet adapter`.

### 2. עדכן את קובץ ה-.env.local

אם ה-IP שלך שונה מ-`192.168.150.118`, עדכן את הקובץ:
```
client/.env.local
```

שנה את השורה:
```
VITE_API_URL=http://YOUR_IP:5000
```

### 3. הפעל את השרתים

```bash
npm run dev
```

### 4. גש מהמכשיר האחר

פתח בדפדפן במכשיר האחר (טלפון/טאבלט/מחשב אחר ברשת):
```
http://YOUR_IP:5173
```

**דוגמה:**
```
http://192.168.150.118:5173
```

## הערות חשובות

- ודא ששני המכשירים מחוברים לאותה רשת WiFi
- אם יש Firewall, ודא שהוא מאפשר חיבורים על פורטים 5000 (שרת) ו-5173 (קליינט)
- אם אתה משנה את ה-IP, תצטרך לעדכן את `client/.env.local` ולהפעיל מחדש את השרת

## פתרון בעיות

**אם לא מצליח להתחבר:**
1. ודא שהשרתים רצים (`npm run dev`)
2. בדוק שה-IP נכון
3. נסה לפתוח את `http://YOUR_IP:5000` בדפדפן - אמור לראות "Glass Dynamic API is Running..."
4. בדוק את ה-Firewall
